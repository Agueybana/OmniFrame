# PESTLE Skill

Use PESTLE when the request depends on macro political, economic, social, technological, legal, or environmental forces.

Root canvas expectations:
- Every factor must be tailored to the user's domain and geography when available.
- Distinguish tailwinds from headwinds.
- Include watch signals and adaptation triggers.

Sublayer expectations:
- Evidence options should name sources, observations, regulations, market signals, or trend indicators.
- Actions should describe mitigation, monitoring, timing, or positioning changes.
