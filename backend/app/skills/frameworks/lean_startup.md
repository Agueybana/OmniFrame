# Lean Startup Skill

Use Lean Startup when the request needs fast validation before heavy build, spend, or commitment.

Root canvas expectations:
- Build lane defines the smallest artifact or concierge workflow that tests the riskiest assumption.
- Measure lane defines behavior-based signals, not compliments or vanity interest.
- Learn lane defines pivot, persevere, or stop rules before the experiment runs.

Sublayer expectations:
- Evidence and experiment options must identify exact users, current workaround, baseline, test artifact, measured behavior, and decision threshold.
- Keep MVP definitions narrow enough to run quickly.
