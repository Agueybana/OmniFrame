# RICE Skill

Use RICE when the request asks what to build, rank, prioritize, sequence, or compare.

Root canvas expectations:
- Infer a feature or initiative list when the user has not provided one.
- Each initiative must be specific to the user's idea and represent a coherent product or execution slice.
- Reach, Impact, Confidence, and Effort must be plausible and internally consistent.
- Score must reflect the displayed numbers.

Sublayer expectations:
- Evidence options justify reach, impact, confidence, or effort with concrete assumptions.
- Regeneration should produce new initiative-specific options, not copies from other score rows.
